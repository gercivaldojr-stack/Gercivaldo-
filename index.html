Funcionalidades

Tudo que voce precisa para converter documentos juridicos

📄

Extracao Inteligente

PDF, DOCX, TXT com tabelas, headers/footers, OCR seletivo por pagina.

🏛

Heuristicas Juridicas

Modo forense e doutrina, citacoes, metadados, pecas processuais.

📊

Deteccao de Colunas

Identifica e reordena PDFs de 2 colunas automaticamente.

⚡

Processamento Paralelo

Multiprocessing para batch e chunks de PDFs grandes.

🌎

Deteccao de Idioma

Auto-detecta pt, en, es, fr, de, it para configurar OCR.

💾

Cache de OCR

Evita re-processamento salvando cache em disco por hash de pagina.

Formatos de Saida

Exporte para o formato que voce precisa

.md

Markdown

Limpo, estruturado, com frontmatter YAML e sumario.

.html

HTML

Documento completo com CSS juridico (serif, blockquotes, tabelas).

.docx

Word / DOCX

Headings, tabelas, blockquotes, listas bullet via python-docx.

Como Usar

Tres formas de executar o conversor

# Conversao basica
python cli.py peticao.pdf

# Com opcoes
python cli.py peticao.pdf -o saida.md --mode forense --toc --citations

# OCR com deteccao automatica de idioma
python cli.py autos.pdf --ocr --ocr-lang auto --ocr-cache

# Exportar como HTML ou DOCX
python cli.py peticao.pdf --format html
python cli.py peticao.pdf --format docx

# Batch paralelo
python cli.py ./autos/ --batch --workers -1

# PDF grande com page range e chunks
python cli.py grande.pdf --pages 1-500 --chunk-size 50

Arquitetura

Pipeline modular em 5 etapas

  PDF / DOCX / TXT
         |
    [1] extract_text()         extractors.py + column_detector.py
         |                       + OCR seletivo + lang_detector.py
         |                       + ocr_cache.py + chunks + parallel.py
         |
    [2] clean_text()           cleaning.py
         |                       dehyphenation, headers/footers, CNJ
         |
    [3] apply_heuristics()     legal_heuristics.py
         |                       headings H1/H2/H3, citacoes, assinaturas
         |
    [4] generate_frontmatter() metadata.py
         |                       YAML + metadados processuais
         |
    [5] export()              html_exporter.py / docx_exporter.py
         |
    .md  |  .html  |  .docx
